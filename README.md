# NES DEV

A NES development playground using ca65 assembler 

## Install cc65

```shell
sudo apt install gcc
sudo apt install make
git clone https://github.com/cc65/cc65.git
cd cc65
make
sudo make avail
```

## Build

Build the ROM file:

```shell
make
```

Clean after build:

```shell
make clean
```

## Install Emulator

```shell
sudo apt install fceux
```
